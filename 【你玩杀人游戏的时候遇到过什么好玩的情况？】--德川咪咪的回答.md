## 你玩杀人游戏的时候遇到过什么好玩的情况？

作者: [德川咪咪](http://www.zhihu.com/people/de-chuan-mi-mi)&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 赞同: 1140


<p><b>很多年以后，当我听到“杀人游戏”四个字，还是会打一个哆嗦，联想到那个冬天的雨夜，我握着手机站在冰冷的雨水中，看着被雨水打花的屏幕，目瞪口呆。</b></p><br><p>我会花很长的篇幅来复盘这场杀人游戏。因为，自那以后，我再也没有玩过线上杀人。</p><br><p>故事从头说起。我是一个记者，擅长观察和质询，记性不差，推理还行。同时，由于运气不错，我很快在魔都杀人游戏群里脱颖而出，一路高歌猛进，获得了五连胜。魔都杀人游戏群的规矩是：每多获胜一轮，便可以在头像前挂一颗星星，挂满5颗，便升级为月亮。</p><br><p>那时，我是场上唯一一个保持不败战绩的玩家，也是最早升级出“月亮”的玩家。与我情况相反的是从未获胜的 <a data-hash="7f5debe4cb9b6f1f5c6b57495f35c8d7" href="http://www.zhihu.com/people/7f5debe4cb9b6f1f5c6b57495f35c8d7" class="member_mention" data-editable="true" data-title="@Hiei" data-tip="p$b$7f5debe4cb9b6f1f5c6b57495f35c8d7">@Hiei</a>  （下称黑黑），他连败5场，嚎啕大哭一场后，在头像前面挂了一个骷髅。</p><br><p>随后我们开了第六场，也就是之后我要说的，我的最后一场、一波三折、刻骨铭心的杀人游戏。</p><br><br><p>场上共有10名玩家，4个角色：2杀手、2警察、5平民和1间谍。间谍，是我们为了线上杀人的平衡性而特意推出的新身份，间谍混在平民阵营中，孤身作战，警察无法验出间谍的身份，杀手也不知道谁是间谍。间谍的任务是保证杀手方获胜，这个身份的能力是，在一轮游戏中，仅有一次开枪的机会，如果打中警察，警察死亡，如未能打中警察，则无伤亡；此外，间谍无法被杀死（如杀手杀中间谍，当晚则为“平安夜”）。</p><br><p>第六场的法官叫阿菊。游戏开初，我的微信震了一下，发现自己被拉进了一个群，抬头一看，群名为“杀手”。</p><br><p>噢，这一轮我是杀手。</p><br><p>“我要赢了。”我还没来得及说话，我的杀手同伴 <a data-hash="20366583aa36670a6aab34635a37bcfd" href="http://www.zhihu.com/people/20366583aa36670a6aab34635a37bcfd" class="member_mention" data-editable="true" data-title="@陈诺" data-tip="p$b$20366583aa36670a6aab34635a37bcfd">@陈诺</a>  先开了口，欣喜若狂，“我终于和五连胜的咪咪分在了一组！这下要躺赢了！”</p><br><p>而我的心情则远不如陈诺轻松。在杀人游戏时，我唯一不擅长的技能点是“伪装”，因此最害怕扮演杀手。而五连胜无疑将我推上了“首杀首验”的风口浪尖——无论是警察还是杀手，都会首先选择对我下手。</p><br><p>此时是上午11点。杀人群中寂静无声，自现在起到下午1点前，每个玩家最多能发言3次，当然也可以选择不发言。</p><br><p>所有人都在谨慎地沉默着，这使我有些微微焦躁。记得先前某一轮，我是通过一个微小的细节发现杀手的——在一名警察已被杀死、另一名警察已暴露的情况下，我在群里发了一个红包说：“杀手干得漂亮，杀手拿。”结果，做贼心虚的杀手成为场上唯一一个不敢拿红包的人，以此为由对他进行质疑时，杀手因为过于慌张，拿出了一大堆截图和证据，说自己当时没有看到这个红包。最终，他引发了怀疑并被投死，一个红包扭转了本来必败的局面。</p><br><p>所以啊，这是一个多么拼、多么残酷、多么excited的游戏。</p><br><p>回到游戏中，第一个开口说话的是 <a data-hash="763553703ff1939cd1988f3ccec666e1" href="http://www.zhihu.com/people/763553703ff1939cd1988f3ccec666e1" class="member_mention" data-editable="true" data-title="@新右门卫" data-tip="p$b$763553703ff1939cd1988f3ccec666e1">@新右门卫</a>  ，他的第一条发言是这样的：“德川你不应该是五连胜吧，我记得之前有一轮规则还不完善，说了是测试轮，你怎么能把这一轮也算进去呢？”</p><br><p>第二个说话的是马爷，她是一个性情豪爽、容易激动的妹子：“这一轮我们新加了间谍这个身份进去，使跳警这个行为变得非常危险。所以我希望大家都能多说几句，因为杀手越沉默，越不容易露马脚。我现在怀疑黑黑、枪叔和新右，请你们都站出来说几句。最后，重要的话重复三遍：在这个规则下，不要找警察、不要找警察、不要找警察。”</p><br><p>“咪咪，我记得之前几轮你都是抢着发言的啊，这轮怎么就沉默了？不好意思，我希望你能够提供一些站得住脚的怀疑和推理，否则你是杀手的嫌疑就会成倍增加。”一个叫梯田的玩家开口了。</p><br><p>我紧张得手心冒汗：心虚，是杀手唯一难过的心理关。一般来说，场面上最少质疑别人的人，往往就是杀手：因为，只有杀手知道别人都是清白的。因此所有杀手都希望遇到一个互相乱咬的混乱场面，并在这个场面中尽量保持沉默，伪装成良民，刷低自己的存在感。而我，由于之前的连胜过于高调，以及胜出后总是忍不住抖机灵，已经不能指望用沉默来伪装了，必须硬着头皮站出来解释。</p><br><p>“我觉得是新右。1点前的3次发言是很宝贵的，他却用了一次发言来跟我抬杠，还在说这种无关紧要的头衔的事情。他显然别有用心。”我说。</p><br><p>转头回到杀手群，我提醒陈诺警惕梯田：“之前没和他玩过，但这人的质疑很有水平，要小心。”</p><br><p>又有几个玩家发了言，随后轮到枪叔。</p><br><p>枪叔是一个神奇的玩家，在我写这个故事时，他特意嘱咐过我，不要透露他在知乎上叫 <a data-hash="a3183ce0664ac190c8ad28147ba49b5a" href="http://www.zhihu.com/people/a3183ce0664ac190c8ad28147ba49b5a" class="member_mention" data-editable="true" data-title="@shotgun" data-tip="p$b$a3183ce0664ac190c8ad28147ba49b5a">@shotgun</a>  。枪叔是做信息安全出身的，看过成吨的推理小说，能言善辩、口才极佳，此外又长着一张智慧无穷的长者脸，使他在杀人游戏中如被忽悠之神附体。滔滔不绝一通，甚至可以颠倒黑白，曾有一名警察被杀后留遗言说验出枪叔是杀手，他竟然硬生生地把那个警察说成是“自杀以构陷我的杀手”，引导平民投死了另一个警察。</p><br><p>可以说，在杀人群中，枪叔是唯一一个存在感比我更强、而且强多了的玩家。</p><br><p>“枪叔，你可以说话了，继续忽悠呀。但是这一次我不会再随随便便信你了，呵呵。”陈诺说。大家都知道，她先前几轮的属性都是百分之百被枪叔忽悠。</p><br><p>“哈哈哈，反正现在我说啥你们不信我。我可能是杀手，也可能是警察，也可能是平民，信不信随你们。说完了，再见。”枪叔说完这句，真的消失了。</p><br><p>法官阿菊提示，下午1点快到了。唯一还没有发言的玩家叫表妹，她是个中学物理老师，周末常和我们一起玩密室。因为颜美声音甜，硬是被新右认作了表妹，拉进群里一起玩。表妹匆匆发了个言，说自己上午课多无暇说话，但一直在爬楼，之后会做出推理。</p><br><p>下午1点是个关键时间：杀手杀人，警察验人，间谍也要在此时选择开不开枪。</p><br><p>“杀谁？”陈诺问我。</p><br><p>“看不出来……”我说。根据目前的发言，我真的看不出谁是警察，陈诺提议说杀高玩，但我拒绝了，“不如杀一个酱油吧，杀了表妹，反正她工作忙没办法全心投入游戏。”</p><br><p>下午1点以后的发言很关键，法官会在下午4点公布死者，但杀过人的杀手和验过人的警察都已心中有数。一些关键的线索将在此刻暴露。</p><br><p>尽管依然只有3次发言机会，但讨论却激烈起来了。每个人都长篇累牍地发言，抓着别人的一点点话柄，大肆发表自己的怀疑。而讨论的焦点变得集中起来：新右和马爷开始互咬。</p><br><p>这种局面对于杀手来说自然大好，我和陈诺幽幽看戏，不时说两句不轻不重、模棱两可的质疑。</p><br><p>4点到了，法官宣布死者：表妹出局。表妹留下遗言：“如果我死了，可能是有人要嫁祸给表哥（新右）吧。”</p><br><p>“新右，果然是你。”马爷听到消息后迫不及待地发言，“我们根本不认识这个表妹，她是你这一局才拉进来的。游戏进行到现在，她几乎一句话都没说，除了你谁会杀她？”</p><p>新右显然已经黑脸了，他更激烈地反驳马爷：“你没听表妹的遗言吗？杀手杀她是为了嫁祸于我好吗？你现在一直咬着我是什么意思，我强烈怀疑你就是杀手，不，我现在已经确定了。”</p><br><p>“我觉得新右这话有问题，在我看来，表妹就是个不明真相的平民。她的遗言不能作数，倒是要警惕新右借表妹的遗言洗白自己。”我说。</p><br><p>“咪咪你是什么意思？你这话很值得怀疑，你为什么随随便便就排除了表妹是警察的可能性呢？毕竟，她是死者，是场上我们唯一可以信任的人。”本来已经消失的枪叔突然发言了。</p><br><p>“枪叔你是在逗我吗？警察会这么留遗言吗？而且我发现枪叔你从刚刚开始就很反常，一直不说话，偶尔说一句还逻辑混乱。这不是你平时的水平，你是杀手吧？”我说。</p><br><p>“呵呵呵，我就知道说什么你们都不会信我。我还是消失吧。”他又消失了。</p><br><p>从公布死者的下午4点至晚上7点，每个人依然只能发言三轮。7点后自由辩论，8点投票。我暗暗等待着自由辩论的时刻，尽管我不知道枪叔是什么身份，但他的怪异举止，显然成为了最好的挡刀肉盾。</p><br><p>果然，7点过后，枪叔已成为场上最大嫌疑人。除了新右紧咬马爷不放，众人纷纷表示，枪叔你再不自证清白，我们可是要投你了。</p><br><p>“好吧。”消失已久的枪叔又出现了，悠然道，“我跳警，我是警察，所以别投我。”</p><br><p>群里明显沉默了十几秒，随后炸了。“有你这么跳警的吗？”“你太假了。”“那你说说你验了谁？”而萌妹子 </p><br><p>“你们会输的。”出局的新右留下遗言，此后他不能、也没有再说过一句话。</p><br><p>阿菊宣布，自现在到次日下午1点，每个人最多能发言3次。本来像开水一般沸腾的群骤然安静。这种诡异的气氛持续了20分钟，马爷和梯田先后发言，他们的情绪都很激动，发言的大意是，枪叔显然不是警察，为什么我们刚才不把他投死？现在想知道大家的看法，尤其是纽太普的解释，为什么只有你跟着枪叔的建议投了椰子……</p><br><p>“梯田应该是警察。”我在杀手群里发表推理，“他在群里话不多，很少随便质疑人。刚才枪叔假跳警，梯田心知肚明，苦于我们局里有间谍，跳警反对太过危险。现在他有点熬不住了。”</p><br><p>“那你觉得另一个警察是谁？”陈诺问。</p><br><p>我飞快地向上翻着聊天记录，然后作出推理：“另一个警察应是马爷。除了梯田，马爷几乎质疑过全场所有玩家。更重要的是，马爷在今天下午1点前曾质疑过黑黑，之后就再也没有质疑过他，我怀疑她和梯田这一轮可能验了黑黑。”</p><br><p>我在杀手群打字打得正欢，来接我下班的纽太普突然凑过头来问：“你觉得谁是杀手啊？”</p><br><p>我吓得差点把手机扔掉，然后愤怒地看着他说：“你干嘛随随便便看我手机？”</p><br><p>“你那么紧张干嘛？难道是你杀手？”他做出名侦探兔美的表情来。</p><br><p>“我怎么可能是杀手？我倒是觉得你很可疑。”我强装镇定，努力掩藏自己的熊吉脸。</p><br><p>“那你来分析分析局势。”</p><br><p>“我只是强烈地感觉马爷是警察。”</p><br><p>“我也觉得她像。”</p><br><p>此后无话，我们心怀鬼胎地分手，各自回家。</p><br><br><p>第二天下午1点，我和杀手同伴陈诺交换了一下意见，很快达成共识：今天先杀梯田，明天再杀马爷。此时，枪叔仍然疯疯癫癫，除了我顾着他的间谍身份，偶尔说两句分散大家的注意力，众人对他的质疑越来越集中。</p><br><p>而一向沉默寡言的梯田，在大约三点多的时候，突然发言：“我越来越怀疑咪咪了，她一会儿质疑这个，一会儿质疑那个，我看她是想通过这种方式把自己洗白。”</p><br><p>“看来梯田这轮是验到我了哦。”我在杀手群说。</p><br><p>“那怎么办？”陈诺问。</p><br><p>“没事，我现在装傻，不回应他的质疑就是了。4点一到，这个梯田就去西天了。这一轮开始没有遗言了，他都不能跳死警指控我。到时候随便我怎么说，他都没法反驳了。哈哈哈哈哈哈。”</p><br><p>4点到了，阿菊宣布，梯田出局。</p><br><p>“两位小姐，你们的意志已经执行，祝你们游戏愉快。”一直以来尽忠职守、绝不多言一句的阿菊，此时在杀手群里忍不住说了这么一句。我的心里短暂地掠过一丝疑虑，但很快无暇顾及了——游戏群里就在此刻风云突变，所有的玩家都震惊了。</p><br><p>是马爷！第一个警察梯田被杀以后，另一个警察马爷疯了。</p><br><p>——“哈哈哈哈哈哈，这一轮都是我的错，是我玩的不好，是我害梯田被杀了。我和梯田是警察，我本来想，我在明处指控，他在暗处隐藏。可我万万没想到，我这个明警没死，他这个暗警却被早早地认了出来。”</p><br><p>——“枪叔，我现在最后悔的就是昨天没有跳警指认你。我们忌惮间谍，不敢跳警。于是就让你这个杀手硬生生地装警察装到现在。”</p><br><p>——“是我玩得不好，我要向所有的平民说对不起，我让枪叔得逞了，都是我的错。我宣布，从此以后退出杀人游戏，永远不会回来了。我有错，我决定用这个方式惩罚自己。”</p><br><p>——“反正我也无所谓了。这局结束我不会再玩了，求你们不要来小窗我，所有人都不要来劝我。”</p><br><p>阿菊在群里发言了：“马爷你闭嘴。4点到7点期间，每个人只能发言3次。”<br></p><br><p>而马爷根本不理他：“我已经无所谓了啊，你把我踢出去好了。梯田死了以后，杀手已经赢了，你觉得我还会在乎这个3次发言限制的狗屁规定吗？”</p><br><p>阿菊干脆利落地把马爷踢了出去：“马爷藐视法庭，踢出群15分钟。7点以后，所有人都能自由发言，马爷禁言15分钟。”</p><br><p>“我们认输吧。”我在杀手群里说，“为了一个游戏，友尽不值得。”</p><br><p>“不要！”陈诺说，“我之前已经4连败了，就快赶上黑黑了。这局好不容易快赢了，就不能让我也赢一下吗？”见我还有疑虑，陈诺补充了一句：“咪咪，我希望你能全心全意地取胜。你放水的话，马爷也不会开心的。”</p><br><p>她的话暂时地说服了我，然而，自由辩论开始，马爷被解除禁言后，我又于心不忍了。尽管她说的大意仍然是“为什么上一轮没有投死枪叔，为什么让他把梯田杀了”，但语气更加凄惨和激烈，颇有“我不配玩这个游戏”的意味，我看着也不想玩了，如果不是陈诺在杀手群不断地劝说，我几次都已把认输的话给写好了，就差发送。</p><br><p>而枪叔，面对马爷满地打滚失心疯发作依然淡定沉默的枪叔，这时候，发言了……他的发言，把游戏推向了又一个高潮，这段话我至今仍然截图保留在手机中：</p><img
